## 🔧 Техническая реализация

### Структура переводов
```typescript
export const translations: Record<string, any> = {
  uk: {
    // Украинские переводы
  },
  en: {
    // Английские переводы
  },
  ru: {
    // Русские переводы
  },
  // ... другие языки
};
```

### Использование в компонентах
```tsx
const { interfaceLanguage } = useLiveStore();
const t = translations[interfaceLanguage] || translations["uk"];

// Использование с fallback
<span>{t.microphone || "Microphone"}</span>
```

### Адаптивные сокращения
Для мобильных устройств используются сокращенные версии:
- `t.microphoneShort` - "Мик" / "Mic"
- `t.systemShort` - "Сист" / "Sys"
- `t.bothShort` - "Оба" / "Both"

## 📱 Особенности локализации

### Мобильная адаптация
- **Сокращения**: Используются для экономии места
- **Тултипы**: Полные названия в подсказках
- **Консистентность**: Единый стиль сокращений

### Качество переводов
- **Точность**: Технически корректные термины
- **Краткость**: Компактные формулировки
- **Естественность**: Естественные для языка выражения

### Fallback значения
Все переводы имеют fallback на английский язык:
```tsx
{t.someKey || "Default English Text"}
```

## 🚀 Добавление нового языка

1. Добавить язык в `LanguageCode` тип
2. Добавить объект с переводами в `translations`
3. Обеспечить все необходимые ключи
4. Проверить сборку

### Пример добавления языка
```typescript
export type LanguageCode = "uk" | "en" | "ru" | "newlang";

export const translations = {
  // ... существующие языки
  newlang: {
    menu: "Menu in new language",
    // ... все остальные переводы
  },
};
```
