# Мобильная адаптация TopBar

## ✅ Реализованные изменения

### 1. Создан компонент MobileMenu
- **Файл**: `src/components/MobileMenu.tsx`
- **Функциональность**: Меню с тремя точками для мобильных устройств
- **Содержит**:
  - Выбор источника аудио (микрофон, система, оба)
  - Настройка скорости воспроизведения
  - Адаптивный дизайн для мобильных экранов

### 2. Обновлён TopBar
- **Файл**: `src/widgets/top-bar/ui/TopBar.tsx`
- **Изменения**:
  - Добавлено состояние `mobile` для выпадающего меню
  - Компоненты скрыты на мобильных устройствах (`hidden lg:flex`)
  - LanguageModeControl позиционируется левее на мобильных устройствах
  - MobileMenu отображается только на мобильных устройствах

### 3. Обновлён LanguageModeControl
- **Файл**: `src/entities/settings/ui/LanguageModeControl.tsx`
- **Изменения**:
  - Адаптивное позиционирование (`lg:left-0 lg:right-auto right-0`)
  - На десктопе: выпадающий список слева
  - На мобильных: выпадающий список справа (чтобы не перекрывать мобильное меню)

### 4. Добавлены переводы
- **Файл**: `src/shared/lib/translations.ts`
- **Новые переводы**:
  - `audioSource`: "Джерело звуку" / "Audio Source"
  - `playbackSpeed`: "Швидкість відтворення" / "Playback Speed"
  - `moreOptions`: "Більше опцій" / "More Options"

## 📱 Поведение на разных устройствах

### Десктоп (lg и выше)
```
[Menu] [New] [FontSize] [Language▼] [Speed▼] [AudioSource]        [WindowControls]
```

### Мобильные (менее lg)
```
[Menu] [New] [FontSize] [Language▼] [MobileMenu⋮]              [WindowControls]
```

### Мобильное меню
При открытии показывает:
```
┌─────────────────────┐
│  Audio Source     │
│  [Mic] [Sys] [Both] │
├─────────────────────┤
│  Playback Speed   │
│  0.75x           │
│  1.0x ✓         │
│  1.25x           │
│  1.5x            │
│  2.0x            │
└─────────────────────┘
```

## 🎯 Ключевые особенности

### Настройки по умолчанию
- **Режим**: Перевод (translation) - вместо разговора
- **Источник звука**: Микрофон (microphone) - вместо системного звука
- **Языки**: Английский ↔ Украинский
- **Скорость воспроизведения**: 1.0x

### Адаптивный дизайн
- **Breakpoint**: `lg` (1024px)
- **Скрытие**: `hidden lg:flex` - компоненты видны только на десктопе
- **Позиционирование**: адаптивное для разных размеров экрана

### UX улучшения
- **Минимализм**: Лишние элементы скрыты на мобильных устройствах
- **Доступность**: Все функции доступны через мобильное меню
- **Интуитивность**: Меню с тремя точками - стандартный паттерн

### Локализация
- **Полная поддержка**: Все тексты локализованы
- **Консистентность**: Единый стиль переводов
- **Языки**: Поддержка всех языков интерфейса
- **Новый язык**: Добавлен русский язык (ru)

## 🔧 Техническая реализация

### Tailwind CSS классы
```css
/* Скрытие на мобильных */
hidden lg:flex

/* Адаптивное позиционирование */
lg:left-0 lg:right-auto right-0

/* Только мобильные */
lg:hidden
```

### React паттерны
```tsx
// Условный рендеринг
<div className="hidden lg:flex items-center gap-2">
  {/* Desktop компоненты */}
</div>

// Мобильные компоненты
<MobileMenu
  isOpen={activeDropdown === "mobile"}
  onToggle={(isOpen) => setActiveDropdown(isOpen ? "mobile" : null)}
/>
```

2. Изменить размер окна < 1024px - компоненты скрываются
3. Открыть мобильное меню - работает корректно
4. Проверить позиционирование - списки не перекрывают друг друга
